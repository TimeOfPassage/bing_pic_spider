### 捷克共和国波希米亚-摩拉维亚高地的长耳鸮
> “长耳”猫头鹰> > 日期: 20221016> > 版权: © Ondrej Prosicky/Alamy
   
 树上是谁？这是一只典型的长耳鸮，有时也被称为小角鸮。它是世界上数量最多的鸮之一，种群数量约为200万-550万，广泛分布于北半球。照片中的长耳鸮被拍摄于捷克。它们能适应各种各样的栖息地，包括北极附近。
