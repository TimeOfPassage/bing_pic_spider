### 神秘森林里木桌上的两个微笑的万圣节南瓜
> 来一盏微笑的南瓜灯吗？> > 日期: 20221031> > 版权: © sandsun/Getty Images
   
 南瓜灯最早起源于古代爱尔兰。后来这个习俗传入美国，新大陆的人们用南瓜替代萝卜，在上面雕刻鬼脸，在里面放上蜡烛，演变成了现在的南瓜灯，成为了万圣节的标志性装饰。
