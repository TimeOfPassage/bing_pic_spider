### 秋季的落羽杉，美国佐治亚州
> 美国南部的别样秋景> > 日期: 20221020> > 版权: © Chris Moore/Tandem Stills + Motion
   
 秋叶飘落，秋天已悄然走近。在这个季节，人们会想到法兰绒和灯芯绒，想到南瓜香料和棉花糖。让我们在晴朗的白天和凉爽的夜晚走出家门，去欣赏这尽染层林吧。说到秋叶，有人认为新英格兰的枫树最美，也有人认为洛基山脉的白杨或美国南部的柏树才是最佳。
