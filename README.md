### 叼着蘑菇的红松鼠
> 一只有趣的小家伙> > 日期: 20220905> > 版权: © Michael Quinton/Minden Pictures
   
 看样子，照片里这只生活在阿拉斯加的红松鼠正准备开始享用它的蘑菇。你可能会感到惊讶，因为松鼠不止吃坚果，它们什么都吃，包括蘑菇。尽管有近20%的野生蘑菇对人类有毒（其中1%会直接致命），但由于松鼠体内有特殊蛋白质，许多毒蘑菇对于它们来说也是可食用的。
