 哈尔施塔特村周围山脉里的盐矿，是世界上最古老的盐矿之一，盐曾是这里的经济支柱。如今，这些矿山成了广受欢迎的旅游景点，美丽的童话村庄吸引了大量游客。随着游客越来越多，哈尔施塔特村不堪重负，甚至开始设法限制每天进城的游客人数。
