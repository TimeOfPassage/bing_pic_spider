### 天堂湾，南极洲
> 南极洲> > 日期: 20221130> > 版权: © SinghaphanAllB/Getty Images
   
 世界上最与世隔绝的陆地无疑是南极洲，那里终年酷寒干燥、寒风凛冽。虽然听起来并不宜居，但南极洲却有着令人惊叹的独特风景，尤其是今天这张照片里呈现的天堂湾。南极洲极易受到气候变化的影响，1959年的今天，多国签署了《南极条约》，旨在保护南极自然生态。这一天也被定为世界南极日。
