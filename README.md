### 日本长野县安昙野附近的燕岳山
> 如果可以的话，请拥抱一座山吧> > 日期: 20220810> > 版权: © Joshua Hawley/Getty Images
   
 每年的8月11日是日本的山之日。日本山之日创建于2014年，是日本最年轻的节日之一。由于地处环太平洋火山带，日本将近四分之三的国土是由山脉组成的。日本的四个主要岛屿：本州、北海道、九州和四国都有山脉贯穿其间。
