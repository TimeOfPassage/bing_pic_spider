> 太阳系的第四颗行星> > 日期: 20221127> > 版权: 
   
 火星距离地球约1.4亿英里，围绕着太阳运行，这颗行星也被称为红色星球，是继水星、金星及地球之后距离太阳第四近的行星。几百年来的研究和探索，使得我们对火星的了解仅次于地球。近年来，随着好奇号和毅力号探测器相继在2012年、2021年成功登陆火星，我们对火星的认识更加深入。每年的11月28号是火星节，让我们在今天庆祝探索火星取得的成就吧。
