### 巴斯克海岸世界地质公园的比斯开湾，西班牙
> 隐入大海的龙尾> > 日期: 20221005> > 版权: © Olimpio Fantuz/eStock Photo
   
 西班牙北部巴斯克海岸比斯开湾边，这些没入海水的龙尾状结构，是地球上最独特和最神奇的地质构造之一。这里是地质学家的仙境，也是奇石爱好者的仙境。在巴斯克海岸地质公园导游的带领下，游客可以在这里发现6000万年不间断的地质历史。
