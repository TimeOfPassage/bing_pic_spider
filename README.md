### 古瓦哈提的竹子，印度阿萨姆邦
> 绿意盎然> > 日期: 20221204> > 版权: © Page Bazar/500px/Getty Images
   
 古瓦哈提位于雅鲁藏布江河岸和西隆高原山麓之间，是印度东北部最大的城市。古瓦哈提地区拥有多种野生动物，包括亚洲象、蟒蛇、老虎、犀牛、野牛、灵长类动物等稀有动物 和濒临灭绝的鸟类。
