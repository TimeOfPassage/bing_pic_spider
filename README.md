### 迪纳利国家公园里的两只驼鹿，美国阿拉斯加州
> 驼鹿发情的季节> > 日期: 20221012> > 版权: © Yva Momatiuk and John Eastcott/Minden Pictures
   
 经过数月的进食、漫游和躲避捕食者，阿拉斯加驼鹿会在秋季的发情期（8月底至10月初）进行繁殖。雄性驼鹿会在树上标记它们的气味来吸引雌性驼鹿，并与其他雄性打架争夺交配权。
