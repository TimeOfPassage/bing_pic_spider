### 阿拉斯加州海恩斯附近的秃鹰群，美国
> 秃鹰的集会> > 日期: 20221111> > 版权: © Marion Vollborn/Minden Pictures
   
 每年11月，多达4000只秃鹰会聚集在阿拉斯加海恩斯附近的奇尔卡特河上，捕食在河边产卵后不久就死亡的鲑鱼。对于鸟类来说这是一场盛宴，美味的鲑鱼源源不断。而对于观鸟者来说，这也是一场视觉盛宴。届时，这里还会举办阿拉斯加秃鹰节，包含猛禽展示、猛禽科普、徒步和现场音乐会等活动。
