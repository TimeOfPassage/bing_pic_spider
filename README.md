### 哈尔基岛的圣约翰骑士中世纪城堡遗址，希腊
> 爱琴海之巅> > 日期: 20221113> > 版权: © Massimo Ripani/eStock Photo
   
 哈尔基，一个位于爱琴海的希腊小岛，是个世外桃源般的地方。当游客蜂拥至其他希腊岛屿时，哈尔基却显得安静而悠闲。从罗德岛搭乘渡轮是登上哈尔基岛的唯一途径。哈尔基岛总面积只有11平方英里，是多德卡尼斯群岛中有人居住的岛屿中面积最小的。
