### 洛桑日内瓦湖上空的暴风雨，瑞典
> 一个黑暗的暴风雨之夜> > 日期: 20221027> > 版权: © Suradech Singhanat/Shutterstock
   
 如果日内瓦湖上空乌云密布的景象让你的心情阴郁沉闷，那么你便可以体会玛丽·雪莱在构思《弗兰肯斯坦》时的心境了，这个故事后来也成为了她的开创性著作。虽然玛丽·雪莱曾写过一些历史小说和旅游书籍，但还是这个描写怪诞生物的恐怖故事让她声名大噪。
