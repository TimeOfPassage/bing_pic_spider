### 乞力马扎罗山附近的大象，安波塞利国家公园，肯尼亚
> 象群在路上> > 日期: 20221203> > 版权: © Diana Robinson Photography/Getty Images
   
 今天是世界野生动物保护日，让我们和象群们一起散散步。照片里这些非洲象正缓步穿过肯尼亚安波塞利国家公园的大草原，远处是乞力马扎罗山。由于栖息地丧失和偷猎，非洲象数量锐减。
