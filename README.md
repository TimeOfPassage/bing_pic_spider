### 秋天的图林根森林与瓦特堡城堡，德国
> 等等，这山上得有座城堡！> > 日期: 20221018> > 版权: © ezypix/Getty Images
   
 在你欣赏今天照片里德国图林根森林的美景时，请注意远处的城堡。那是瓦特堡，建于中世纪。马丁·路德流亡期间就在瓦特堡将《新约》翻译成了德语。它是该地区最受欢迎的旅游胜地，并因其典型的中世纪建筑特色而闻名于世。
