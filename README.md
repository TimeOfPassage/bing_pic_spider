### 清迈易鹏节升起的天灯，泰国
> 祝福和祈愿的日子> > 日期: 20221107> > 版权: © Suttipong Sutiratanachai/Getty Images
   
 每逢泰国的易鹏节，放飞天空的孔明灯就如点点繁星，映衬着那一轮圆月。今天的照片展示的就是人们在清迈放飞孔明灯、庆祝易鹏节的盛景。清迈是泰国第二大城市，也是易鹏节的发源地。
