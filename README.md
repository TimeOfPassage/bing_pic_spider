### 萨塞克斯郡的西欧刺猬，英国
> 可以观赏，请勿触碰> > 日期: 20221108> > 版权: © Jules Cox/Minden Pictures
   
 今天照片里的是一只西欧刺猬，拍摄于英国萨塞克斯郡，不过其实西欧刺猬已经遍布整个欧洲大陆。它们被认为是刺猬家族中体型最大、体重最重的种类。刺猬是一种夜行动物，它们经常在花园里活动，品尝它们喜欢的昆虫小食。
