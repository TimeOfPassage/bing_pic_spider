### 北美水獭在阿卡迪亚国家公园游泳，美国
> 北美水獭> > 日期: 20221114> > 版权: © George Sanker/Minden Pictures
   
 今天照片里的是两只北美水獭，拍摄于阿卡迪亚国家公园。严格来说，这些别名“水狗”的水獭是黄鼠狼家族的成员。它们不只出现在河里，还遍布北美的水道和海岸。水獭眼睛近视，视力适合辅助游泳，因此它们有时会冲向人类或船只，可能并非故意，只是没看见。
