### 大沙丘国家公园和保护区，美国科罗拉多州
> 如沙漏中的沙子一般> > 日期: 20220912> > 版权: © Y Paudel/Getty Images
   
 大沙丘国家公园及保护区拥有750英尺（甚至更高）的沙丘，覆盖面积超过30平方英里。但是这些北美最高的巍峨沙丘只是科罗拉多州众多惊人景观之一，这里还有针叶林、高山湖泊和湿地。这个面积为85000英亩的公园及保护区里，甚至还有高山冻原在桑格雷-德克里斯托山脚下蔓延。
