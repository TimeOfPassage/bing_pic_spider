### 罗兰豁口上空的银河，法国上比利牛斯省
> 从天而降的魔法> > 日期: 20220914> > 版权: © SPANI Arnaud/Alamy
   
 如此美景在前，让人惊叹不已。这是罗兰豁口，位于上比利牛斯省的加瓦尔尼冰斗，豁口的那头流淌着星光灿烂的银河。这个豁口是天然形成的，宽40米，深70米，深受徒步旅行者的欢迎。它位于海拔2807米的地方，扮演着山口的角色，也标示着西班牙和法国之间的边界。
