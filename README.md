### 冰川国家公园的山羊，美国蒙大拿州
> 它们对你也同样好奇> > 日期: 20221216> > 版权: © Sumio Harada/Minden Pictures
   
 比起美国其它48个州，你更可能在蒙大拿州的冰川国家公园里看到雪羊。雪羊已经习惯人类了，会悠闲地走到游客面前。不过，还是建议你跟它们保持距离，因为雪羊的“铁头功”可不是开玩笑的。
