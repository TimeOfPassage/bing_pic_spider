### 维尔登的野生动物通道，荷兰
> 谁在使用这座绿草如茵的桥？> > 日期: 20220831> > 版权: © Frans Lemmens/Alamy
   
 过马路的不止有鸡，还有青蛙、麋鹿等等许多野生动物。当穿越人类活动区与野生动物栖息地之间的数千条高速公路时，这些动物不得不面临生死难料的命运。可悲的是，每年有数百万只动物的命运并不如意。而动物与车辆相撞，还会导致司机的死亡。
