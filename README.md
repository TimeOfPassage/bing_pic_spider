### 菲斯皇宫的大门，摩洛哥
> 通往皇宫的大门> > 日期: 20221117> > 版权: © Adam Smigielski/Getty Images
   
 2019年，联合国教科文组织宣布将每年的11月18日定为“伊斯兰艺术国际日”，旨在宣传古代和现当代伊斯兰艺术，以及伊斯兰艺术对人类文明的贡献。11月18日也是摩洛哥的独立日，纪念摩洛哥苏丹穆罕默德五世1955年结束流亡回到摩洛哥。
